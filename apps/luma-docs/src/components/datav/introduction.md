# DataV 组件概览

来自 `@luma/datav`。支持 DataV `config` 与现代 props；优先级：显式 props > config > 默认。

```bash
pnpm add @luma/datav vue echarts
```

```ts
import LumaDatav from '@luma/datav'
import '@luma/datav/style.css'
app.use(LumaDatav)
```

或按需：`import LumaBorderBox from '@luma/datav/border-box'`。

- 示例长文：[DataV 示例](/reference/datav-examples)
